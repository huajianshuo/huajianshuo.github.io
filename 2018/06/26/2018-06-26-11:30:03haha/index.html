Jun 26, 2018 1991

#

1.

老师；今天下午在我幼小的身体里发生了一件吓人的事情，我体内的横幅机忽然在一种神秘的力量下有规律的颤动，频率是每三秒一次，强度是8·3级，浑身一颤并带有一声鸡叫似的声音，望老师准假，到医院检查检查是什么病。老师读完纸条评语‘打嗝就打嗝吧，说成你这样的，真是没谁了。’

#

2.

今天借了表妹的校服穿着卖萌，表妹说；‘姐，听说你们90后都到晚婚年龄了？’听完我默默地把校服还给表妹，然后把她早恋的事告诉了小姨妈。

#

3.

最近养了一对小兔子，老妈嫌弃有味道，就让我在楼道里，放个小笼子养着。我怕它乱吃别人给的东西，就写了张字条“请勿往笼子里投食”结果第二天我发现兔子没了，就剩一张字条，上面写着“好的.我抱回家里喂了”

#

4.

A：这是2天前向借你的2000块钱，现在还给你。
B：没事兄弟，我这几天不要急用钱，你先拿着吧。
A：拿着吧，有钱就应该还啊，有借有还再借不难嘛。
劫匪：你俩人BB啥那，快点，把身上的钱都交出来。

#

5.

两位女生下晚自习后，
女生1：“听说这条路不太安全，我有点害怕。”
女生2：“听说是不太安全，不过也没什么好怕的，这条路离家近。”
女生1：“咱们还是走别的路吧！真的不安全！”
女生2：“没关系的。即使这条路不安全，咱们俩也长得很安全！”

#

6.

领导买了一箱牛奶放自己办公室，然后发现丢了好几盒。中午吃饭时，领导语重心长地说了这个事，希望偷喝的人能主动承认错误并还回来，末了加了一句:“其实箱子上是可以查到指纹的。”等下午领导再回去，连箱子都不见了……

#

7.

就刚刚，哥们我上公交时前面一漂亮妹子发夹掉了，我捡起上前叫住她：“美女你发夹掉了。”妹子微微一笑，伸手来接。从没跟妹子搭讪过的我不知哪来的勇气，我竟然说：“能认识下吗？交个朋友。”结果，妹子华丽丽地来了句：“算了，发夹我不要了。

#

8.

在餐厅吃饭时，小强和服务员发生了点争执。气得小强夺门而出，谁知那服务员得理不饶人，竟然追了出来。边追还边骂道：“你特么把门给我放下！”

#

9.

我和一个同学去乡村看电影，因为那时有点晚，人都到齐了，我说：现在可以进去吗？检票员说：不行！我问：为什么???他小声的说：这部片子太烂，我怕一开门就全跑出去了

#

10.

我和儿子有个共同的心愿，出国旅游。昨天儿子考试得了全班第一，我跟媳妇合计着带他出国见见世面，吃晚饭的时候。。我：“儿子，你的心愿是什么？”儿子：“吃汉堡包”我：“往大了说。”儿子：“变形金刚”我：“今天你爹说了算，想想，咱俩共同的心愿。”儿子怯生生的瞅了媳妇一眼说：“换个妈？

#

11.

为什么许多海绵宝宝的骨灰粉都只说海星淹死在海里和海里生火不科学却没有一个说谢老板和虾霸是熟的啊

#

12.

小逗采访精神病院院长，….
问：“怎样确定病人是否治愈？”
院长：“其实很简单，把浴缸注满水，旁边放一把汤匙和一个脸盆，看他怎么把浴缸淘空。”
小逗惊奇：
“这也太简单了！我都不用想就知道该用脸盆。”
院长语塞5秒，
答：“正常人和已治愈的人会把浴缸里的塞子拔掉。”

#

13.

话说，把硬东西嚼碎了在喂给别人是一种爱，我可不认为，因为小时候父母就喜欢把甘蔗嚼碎了再喂给我

#

14.

爸爸很懒，每次口渴都让两岁多的小宝贝去拿水，宝贝很乖，总是听话的用杯子端来给爸爸喝。每次爸爸都很得意的望着孩子一饮而尽！妈妈在旁边看着，一直默不作声。
有一天妈妈突然开口问爸爸：“你有没有想过？”“想过什么？”“整个屋子里，他唯一能拿到水的地方是哪里？
（爆笑哥，能通过吗？）

15.

医生：最近多开窗通风
我：外面雾霭太严重了
医生：那就给家里换点新鲜的雾霭。

#

16.

奶奶养了十多年的狗死了，很伤心。为了安慰奶奶，我不知怎么想的，夜里开始汪汪汪地学起狗叫来。后来奶奶找人对我进行了驱魔仪式。

#

17.

老婆：“今天我们在办公室讨论谁的老公最好，结果你胜出了。”
老公笑道：“是吗？她们怎么夸我？”
老婆说：“她们都说长得丑男普遍善良，没本事的男人才会怕老婆。”

#

18.

有个室友养了两只乌龟，喂食的时候总是喊：“来来来兄弟们吃饭了。”
我就嘲笑他：“怎么跟乌龟称兄道弟？”
他严肃的说：“我和它们是结拜兄弟，不求同年同月生，但求同年同月死。”

19.

舍友跟宿舍里放了个臭屁，还在那边狡辩：这不叫放屁！这是菊花在呼吸！
我：你这光呼了，没吸啊！
舍友：你们不是吸了么！

20.

一天夜晚，小丑鱼要睡觉了，小丑鱼爷爷来陪伴着它。
小丑鱼说：“爷爷！爷爷！可以给我讲一个恐怖故事吗？”
爷爷愣了愣：“你不怕？！”
小丑鱼挺起胸膛：“嗯！不怕！”
爷爷神色不安的拿起菜谱：“辣椒蒸鱼头的做法：买一条鱼，将其头部切下，拿来泡椒，切好备用，红辣椒，蒜末都切好备用，下锅翻炒，蒸煮20分钟即可完成…………”
小丑鱼晕了过去……
